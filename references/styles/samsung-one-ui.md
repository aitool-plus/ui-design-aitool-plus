## 1. 设计哲学

### 1.1 三大核心原则

| 原则 | 英文 | 含义 |
|------|------|------|
| 舒适 | Comfort | 单手操作优先，核心交互集中在下半屏，减少手指伸展疲劳 |
| 清晰 | Clarity | 信息层级分明，重点内容突出，视觉降噪减少干扰 |
| 凝聚 | Cohesion | 跨设备统一体验，手机/平板/手表/PC视觉语言一致 |

### 1.2 设计准则

- 单手操作优先：核心操作区域集中在屏幕下半部分，顶部仅展示信息
- 双区布局：上半屏展示标题与概览信息，下半屏承载操作与详细内容
- 折叠屏适配：内屏/外屏/Flex模式三种形态无缝切换
- Samsung生态：与Galaxy Watch、Galaxy Buds、Galaxy Tab跨设备体验连贯

### 5.2 大圆角设计原则

- One UI采用大圆角设计（16-28dp），营造柔和舒适的视觉感受
- 圆角与组件尺寸成正比：小组件8-16dp，中等组件16-24dp，大组件24-28dp
- 折叠屏内屏圆角可适当增大至28-32dp，匹配设备圆角

## 6. 阴影与层级

### 6.1 层级定义

| 层级 | Elevation | 阴影参数 | 用途 |
|------|-----------|---------|------|
| Level 0 | 0 | 无阴影 | 背景层、列表项 |
| Level 1 | 1 | 0dp 1dp 4dp rgba(0,0,0,0.08) | 卡片、分组 |
| Level 2 | 2 | 0dp 3dp 8dp rgba(0,0,0,0.12) | 浮动按钮 |
| Level 3 | 3 | 0dp 6dp 16dp rgba(0,0,0,0.14) | 弹窗 |
| Level 4 | 4 | 0dp 10dp 24dp rgba(0,0,0,0.16) | 模态弹窗 |

### 6.2 柔和扩散阴影

- One UI阴影采用柔和扩散效果，阴影边缘模糊范围大
- 阴影偏移量小（1-10dp），扩散范围大（4-24dp）
- 深色模式阴影效果极弱，使用亮度差替代阴影区分层级

### 6.3 深色模式阴影调整

- 深色模式阴影色值：rgba(0,0,0,0.30) ~ rgba(0,0,0,0.50)
- 深色模式卡片使用1dp边框替代阴影：1dp rgba(255,255,255,0.06)

## 10. 设计禁忌

### 10.1 绝对禁止

1. **禁止小触控区域**：所有可交互元素触控区域不得小于48dp，确保单手操作舒适
2. **禁止顶部密集操作**：核心操作按钮不得放置在屏幕上半部分，必须集中在下半屏
3. **禁止使用非Samsung One字体**：UI界面必须使用Samsung One，品牌展示区除外
4. **禁止小圆角设计**：卡片/弹窗等组件圆角不得小于16dp，保持柔和视觉
5. **禁止忽略折叠屏适配**：必须同时设计内屏/外屏/Flex模式三种布局
6. **禁止自定义系统控件**：Switch、Slider、Progress等系统控件不得替换为自定义实现
7. **禁止使用固定色值代替语义色**：必须使用系统语义化颜色，不得硬编码色值
8. **禁止忽略深色模式**：所有界面必须同时设计浅色/深色两套方案

### 10.2 强烈不建议

1. 不建议在深色模式下使用纯白(#FFFFFF)背景
2. 不建议在屏幕顶部放置主要操作按钮
3. 不建议使用线性动画替代弹性动画
4. 不建议忽略双区布局原则（标题在上/操作在下）
5. 不建议在弹窗中使用关闭按钮替代手势关闭
6. 不建议忽略无障碍对比度要求（文字与背景对比度不低于4.5:1）

### 15.2 模糊效果性能建议

| 参数 | 建议值 | 说明 |

| 推荐模糊半径 | 20dp | 系统默认 |
| 实时模糊区域 | 不超过屏幕30% | 大面积模糊降级 |
| 替代方案 | 半透明背景色 | 低端设备降级 |

### 15.3 阴影渲染建议

| 建议 | 说明 |
|------|------|
| 静态阴影 | 使用elevation属性 |
| 动态阴影 | StateListAnimator管理 |
| 深色模式 | 亮度差替代阴影 |
| 阴影数量 | 单屏动态阴影不超过6个 |

### 15.4 图片加载策略

| 加载阶段 | 策略 | 说明 |
|---------|------|------|

| 低清图 | 缩略图(宽200px) | 优先加载 |
| 高清图 | 原图 | 后台加载替换 |
| 缓存 | 双缓存 | 内存+磁盘 |

### 15.5 列表虚拟滚动建议

| 参数 | 建议值 | 说明 |
|------|--------|------|

### 15.6 重绘优化建议

| 规则 | 说明 |
|------|------|
