## 1. 设计哲学

### 1.1 三大核心原则

| 原则 | 英文 | 含义 |
|------|------|------|
| 人车家全生态 | Human x Car x Home | 手机/汽车/智能家居全场景互联，统一体验 |
| 年轻化 | Youthful | 高饱和度色彩、活力动效、信息密度高，面向年轻用户 |
| 高信息密度 | High Information Density | 单屏展示更多有效信息，减少操作层级，效率优先 |

### 1.2 设计准则

- 生态流转：应用/服务在手机、车机、智能家居间无缝流转
- 功能卡片化：核心功能以卡片形式前置，信息密度高
- 紧凑高效：间距紧凑、信息密集，单屏承载更多内容
- 年轻活力：色彩饱和度高、动效流畅、视觉节奏快

## 6. 阴影与层级

### 6.1 层级定义

| 层级 | Elevation | 阴影参数 | 用途 |
|------|-----------|---------|------|
| Level 0 | 0 | 无阴影 | 背景层、列表项 |
| Level 1 | 1 | 0dp 1dp 2dp rgba(0,0,0,0.06) | 卡片微浮 |
| Level 2 | 2 | 0dp 2dp 6dp rgba(0,0,0,0.10) | 浮动按钮 |
| Level 3 | 3 | 0dp 4dp 12dp rgba(0,0,0,0.12) | 弹窗 |
| Level 4 | 4 | 0dp 8dp 20dp rgba(0,0,0,0.14) | 模态弹窗 |

### 6.2 卡片阴影

| 卡片状态 | 阴影参数 | 说明 |
|---------|---------|------|
| 静态 | 0dp 1dp 2dp rgba(0,0,0,0.06) | 默认展示 |
| 悬浮 | 0dp 4dp 12dp rgba(0,0,0,0.12) | 长按/拖拽状态 |

### 6.3 深色模式阴影调整

- 深色模式下阴影效果极弱，主要依赖亮度差区分层级
- 深色模式阴影色值：rgba(0,0,0,0.30) ~ rgba(0,0,0,0.50)
- 卡片深色模式使用1dp边框替代阴影：1dp rgba(255,255,255,0.06)

## 10. 设计禁忌

### 10.1 绝对禁止

1. **禁止低信息密度**：功能卡片、列表等组件必须展示充分信息，不得大面积留白浪费空间
2. **禁止老式MIUI风格**：不得使用MIUI 12及更早版本的视觉风格，必须遵循HyperOS设计语言
3. **禁止使用非MiSans字体**：UI界面必须使用MiSans，品牌展示区除外
4. **禁止使用低饱和度配色**：主色调必须保持高饱和度，不得使用灰暗配色
5. **禁止忽略生态流转**：核心功能必须支持跨设备流转，不得绑定单一设备
6. **禁止自定义系统控件**：Switch、Slider、Progress等系统控件不得替换为自定义实现
7. **禁止使用固定色值代替语义色**：必须使用系统语义化颜色，不得硬编码色值
8. **禁止忽略深色模式**：所有界面必须同时设计浅色/深色两套方案

### 10.2 强烈不建议

1. 不建议在深色模式下使用纯白(#FFFFFF)背景
2. 不建议使用小于44dp的触控热区
3. 不建议在功能卡片中使用过多装饰元素
4. 不建议使用线性动画替代弹性动画
5. 不建议忽略控制中心/负一屏组件适配
6. 不建议在弹窗中使用关闭按钮替代手势关闭
7. 不建议忽略无障碍对比度要求（文字与背景对比度不低于4.5:1）
8. 不建议在卡片中使用超过2级信息层级

## 15. 性能与渲染约束

### 15.1 动画帧率要求

| 场景 | 帧率要求 | 说明 |
|------|---------|------|
| 核心交互动画 | 60fps最低 | 不得低于60fps |
| 页面转场 | 60fps | 系统动画 |
| 滚动 | 60fps | RecyclerView流畅 |
| 生态流转 | 60fps | 跨设备动画 |

### 15.2 模糊效果性能建议

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 高斯模糊半径上限 | 25dp | 超过25dp性能下降 |
| 推荐模糊半径 | 20dp | 系统默认 |
| 实时模糊区域 | 不超过屏幕30% | 大面积模糊降级 |
| 替代方案 | 半透明背景色 | 低端设备降级 |

### 15.3 阴影渲染建议

| 建议 | 说明 |
|------|------|
| 静态阴影 | 使用elevation属性 |
| 动态阴影 | StateListAnimator管理 |
| 深色模式 | 亮度差替代阴影 |
| 阴影数量 | 单屏动态阴影不超过6个 |

### 15.4 图片加载策略

| 加载阶段 | 策略 | 说明 |
|---------|------|------|
| 占位色 | 主色调低饱和度色块 | 即时展示 |
| 低清图 | 缩略图(宽200px) | 优先加载 |
| 高清图 | 原图 | 后台加载替换 |
| 缓存 | 双缓存 | 内存+磁盘 |

### 15.5 列表虚拟滚动建议

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 长列表阈值 | 50条以上 | 必须使用RecyclerView |
| 缓存池 | 可见项+2 | RecycledViewPool |
| 分页加载 | 每页20-50条 | 滚动到底自动加载 |

### 15.6 重绘优化建议

| 规则 | 说明 |
|------|------|
| 避免布局抖动 | 禁止onDraw中修改布局 |
| 减少视图层级 | ConstraintLayout扁平化 |
| 避免过度绘制 | 背景重叠不超过2.5x |
| ViewStub | 延迟加载不可见区域 |
