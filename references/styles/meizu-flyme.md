# 魅族 Flyme 设计规范

## 一、设计哲学

Flyme 的设计哲学围绕「极简克制、大面积留白、统一图标、清新雅致」四大核心理念展开。系统界面追求极致的简洁与克制，通过大面积留白、克制的色彩运用和统一的图标体系，营造清新、雅致、宁静的产品气质。Flyme 设计语言强调「少即是多」，去除一切不必要的装饰元素，让用户专注于内容本身。

### 2.4 配色原则

Flyme 采用低饱和度配色方案，主色调使用魅族蓝，辅助色以灰色系为主。色彩运用克制，强调色仅用于关键操作和状态指示，避免大面积高饱和度色彩的使用，保持界面的清新与雅致。

## 十、设计禁忌

1. **禁止过度装饰**：界面元素必须保持极简，禁止添加阴影、渐变、描边等装饰性效果，除非功能需要。
2. **禁止高饱和度配色**：避免使用高饱和度的色彩，主色调、辅助色均应保持低饱和度，保持界面的清新雅致。
3. **禁止密集布局**：页面必须保持大面积留白，组件间距不得低于 16dp，禁止紧凑、拥挤的布局方式。
4. **禁止大圆角**：组件圆角不得超过 12dp，核心组件保持 8-12dp 适度圆角，禁止过度圆润。
5. **禁止复杂动效**：动画时长不得超过 300ms，禁止弹跳、弹性、旋转等复杂动效，保持动效的克制与优雅。
